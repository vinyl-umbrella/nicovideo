# ニコ動 (Re:仮) 検索サービス (仮)

本 Web アプリケーションでは，[ニコニコ動画 (Re:仮)](https://www.nicovideo.jp/) で公開されている動画の検索を容易にすることを目的としています．

動画の再生やコメントの投稿機能を作成する予定は一切ありません．


## 仕様技術

- フロントエンド: preact
- バックエンド: golang (gin, gorm)
- データベース: MySQL8.4
